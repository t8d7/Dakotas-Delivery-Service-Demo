# Dakota's Delivery Service Demo
Simple GB Studio based demo.

This demo follows Dakota as she opens a potion delivery service in a fantasy-based world. Currently the demo is extremely limited in scope, and has 1 simple delivery. This was created by myself (Thomas Miller) alongside Jiyun Chung with his artwork. Unfortunately there is no music, as neither of us are musically talented, but as this is fully open source, feel free to play with it how you see fit.
